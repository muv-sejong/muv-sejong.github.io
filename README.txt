# MUV Lab Website

## File structure
- index.html: SpaceX-style landing page
- research.html: detailed research page
- projects.html: detailed project page
- publications.html: detailed publication page
- members.html: PI and members page
- join.html: graduate recruitment page
- contact.html: contact page
- style.css: common style
- script.js: mobile menu + count-up animation
- assets/: images and profile photos

## Existing video files
Keep your existing video files in the assets folder:
- assets/hero.mp4
- assets/hero-mobile.mp4
- assets/poster.png (optional)

If hero videos are missing, the hero section will still show assets/research.png as a fallback background.

## Editable numeric counters
Edit data-target values in index.html and publications.html/projects.html:
- Total Published Papers
- SCIE Papers
- Papers in Progress
- Defense / Industry Projects
- National / Public R&D Projects
- Ongoing Major Projects
